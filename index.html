Cloud native media streaming that just works

Gilfoyle is a web application from the Dreamvo project that runs a self-hosted media processing server.

Get started

100% open-source, no vendor lock in. Licensed under GPL-3.0.

Easy to integrate

Client SDKs are here to help the media streaming implementation to be as easy as possible in your application.

Deploy anywhere

Thanks to cloud native methodologies, Gilfoyle can be deployed anywhere, especially on modern cloud infrastructures.

Monitoring

Enjoy built-in Prometheus exporter including a bunch of monitoring metrics that helps keeping your instance healthy.

Agnostic by design

Gilfoyle was built with the vision of serving any technical use case that may exist in the media streaming industry.

100% open source

Gilfoyle is free to use, modify and distribute. We don't practice any kind of vendor lock-in.

It's cheap!

Video encoding can be painful because of performances it needs. Gilfoyle ensure resources are well used and allow you to scale at any time.
