Simplify media streaming workflow

Gilfoyle is the end-to-end solution to easily implement media streaming in any application at any scale.

Get started

100% open-source, no vendor lock in.

Easy to integrate

Client SDKs are here to help the media streaming implementation to be as easy as possible in your application.

Deploy anywhere

Thanks to a cloud native architecture, Gilfoyle can be deployed anywhere, especially on modern cloud infrastructures.

Monitoring

Enjoy built-in Prometheus exporter including a bunch of monitoring metrics that helps keeping your instance healthy.

Agnostic by design

Gilfoyle was designed with the ambition of serving most of technical use cases that may exist in the streaming industry.

100% open source

Gilfoyle is free to use, modify and distribute. We don't practice any kind of vendor lock-in. It's licensed under GPL-3.0.

Adaptative streaming

Thanks to Adaptative Bitrate Streaming (ABR) and HTTP Live Streaming (HLS), Gilfoyle can handle fast video encoding and deliver content in different renditions.
